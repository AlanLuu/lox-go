# String reader methods

String reader objects have the following methods associated with them:
- `stringreader.read(buffer)`
- `stringreader.readAt(buffer, offset)`
- `stringreader.readByte()`
- `stringreader.readBytes(num)`
- `stringreader.readBytesIter(num)`
- `stringreader.readChar()`
- `stringreader.readChars(num)`
- `stringreader.readCharsIter(num)`
- `stringreader.readToFile(file)`
- `stringreader.reset(str)`
- `stringreader.seek(offset, whence)`
- `stringreader.size()`
- `stringreader.sizeutf8()`
- `stringreader.unreadByte()`
- `stringreader.unreadByteBool()`
- `stringreader.unreadChar()`
- `stringreader.unreadCharBool()`
