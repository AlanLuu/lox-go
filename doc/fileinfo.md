# File info object methods

Any method that fails will throw a runtime error with a message describing the error.

The following methods are defined in file info objects:
- `fileinfo.fileMode()`
- `fileinfo.fullName()`, which is an alias for `fileinfo.path`
- `fileinfo.hasNil()`
- `fileinfo.isAppendOnly()`
- `fileinfo.isCharDevice()`
- `fileinfo.isDevice()`
- `fileinfo.isDir()`
- `fileinfo.isNamedPipe()`
- `fileinfo.isRegular()`
- `fileinfo.isSetgid()`
- `fileinfo.isSetuid()`
- `fileinfo.isSocket()`
- `fileinfo.isSticky()`
- `fileinfo.isSymlink()`
- `fileinfo.modeStr()`
- `fileinfo.modTime()`
- `fileinfo.name()`
- `fileinfo.path()`
- `fileinfo.size()`
